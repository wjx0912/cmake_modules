# 版本说明

boost-1.75.0（截至2021.4.25，cmake-v3.20 不支持boost-1.76.0）



# 下载安装

https://sourceforge.net/projects/boost/files/boost-binaries/1.75.0/

FindBoost.cmake里面有：APPEND _boost_INCLUDE_SEARCH_DIRS PATHS  C:/local/boost_${ver}

所以，安装boost到默认路径比较方便
